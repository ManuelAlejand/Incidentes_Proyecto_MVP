# Frontend - Dashboard Gestión de Incidentes

Este directorio contiene la aplicación React que sirve como interfaz del Dashboard.

## 🚀 Inicio Rápido

1. Instalar dependencias:
   ```bash
   npm install
   ```

2. Ejecutar en desarrollo:
   ```bash
   npm run dev
   ```

## 🛠️ Tecnologías
- React + TypeScript
- Vite
- Recharts
- SheetJS (XLSX)
- Lucide React
